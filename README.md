# 🛒 Backend E-Commerce

A Node.js + Express based backend for an e-commerce application built using MVC architecture.
It includes user authentication, product handling, and basic backend features.

---

## 🚀 Features

* 🔐 User Registration & Login
* 🔑 Authentication system
* 📦 Product listing
* 👤 User management (in-memory)
* 📧 Forgot Password (using Nodemailer)
* 🧱 MVC architecture

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* EJS (for views)
* Nodemailer
* JavaScript
